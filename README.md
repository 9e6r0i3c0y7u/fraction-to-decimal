# 分數轉小數練習器

這是一個可直接部署到 **GitHub Pages** 的純前端練習網站，功能包含：

- 題數可調整
- 題型可複選：真分數、假分數、帶分數
- 分母只會出現由 **2 和 5 的因數組成** 的數
- 三種模式：
  - 引導模式（分步填空）
  - 直接轉小數
  - 限時模式
- 限時模式又分成：
  - 一題一題出題，每題限時
  - 一次出完全部，總時間限制

## 檔案結構

- `index.html`：主頁面
- `style.css`：樣式
- `script.js`：出題與作答邏輯

## 本機開啟

直接用瀏覽器打開 `index.html` 即可。

## 部署到 GitHub Pages

1. 建立一個新的 GitHub repository
2. 把這三個檔案上傳上去
3. 到 repository 的 **Settings**
4. 找到 **Pages**
5. Source 選擇部署分支，例如 `main`
6. 資料夾選擇 `/ (root)`
7. 儲存後稍等幾分鐘即可開啟網站
